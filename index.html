LINUX FOR THE PEOPLE!


What is xerOS?

command-line-focused os - a gentle introduction to living in the command line - "sensible" default i3 configuration and a few small tools to help manage a system based on i3 + ubuntu) - it's for developers who want a friendly introduction to using the command line (training wheels, helpful tools for easing into the command line world (it doesn't have to be an overwhelming and scary place) - "elementary OS" for the command line

a "friendly" / helpful human-focused layer to help human beings better interact with machines (but still remain close to the 'metal', not getting lost in superfluous abstractions / kitschy window managers w/ bloated ui)

sane and friendly starting point for command line life - a cool and fun and minimal configuration you don't have to do very much to set up!
  • ubuntu server (currently 21.04)
  • i3 gaps
    • cool default keyboard shortcuts/workflows for i3
    • cool default themes (gtk/i3)
    • ?? fun easy way of editing themes/configs/shortcuts? (maybe part of xeros-utils?)
  • polybar (+cool config?)
  • xterm/urxvt
  • fish (/zsh/bash, fish is default)
    • cool default theme (based on slavic cat)
  • ranger
  • nvm/node/npm
    • (python, rust, ruby, go)
    • dev bundle - popular tools (developer favorites) for working with various languages/technologies
  • micro (/vim/emacs, micro is default editor)
  • htop
  • mosh
  • docker
  • chromium? - bare-bones browser (that's always running, config has scriptable shortcuts/hooks to open new windows)
  • xeros-utils [text gui] / `xh` [cli abstraction helper]
  • workflow "primitives" to make dev processes easy (launching/managing new windows/workspaces, app/focus/workspace switching, helpful hotkeys [and easy system/prefs config editor])
  • ?? text gui (curses) tools for wifi, audio, network, bluetooth, display, system -- primitive support for peripherals, but doesn't go too in-depth w/ bells & whistles (??)
  • ?? other general dev tools (make, idk what else...ubuntu dev tools, lower level libs idk)
  • idk other dev tools (databases? redis/postgres/maria? nginx/apache? xampp? ngrok1/tunneller?)
a pre-configured "development workshop" where all the tools are wired up and ready to go (baseline OS for starting your projects, all the BIG stuff is already set up for you)
most importantly: it looks cool as shit and it's fun to use!
It's for :
  • people who want a gentle introduction into the command line world (noobs, curious)
  • expert users who want to fuck with stuff but don't want to waste a lot of time setting up something simple that ~just works~ [want a good starting point]
